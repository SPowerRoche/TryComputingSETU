# Other Methods to use

**What Are Built-In Methods in Processing?**

Processing comes with hundreds of pre-made methods that you can use without writing them yourself.

They are part of the Processing language and handle:

- drawing shapes

- setting colours

- controlling the canvas

- working with the mouse/keyboard

- animation (frameCount, redraw, etc.)

When you call a built-in method, you pass parameters — pieces of information that tell Processing how to do something (size, colour, position, etc.).

**Example:**

~~~java
    ellipse(100, 200, 50, 80);
~~~

ellipse is the method.

The numbers are the parameters:

    - x position
    - y position
    - width
    - height

## Some you can add to your code

**size(width, height)** 

Sets the size of your canvas.

~~~java
size(400, 400); //400 pixels wide by 400 pixels high
~~~

**background(value)** or **background(r, g, b)**

Sets the background colour.

~~~java
background(220);          // grey - One value → greyscale
background(100, 200, 255); // blue - Three values → RGB colour
~~~

 
**ellipse(x, y, width, height)**

Draws a circle or oval.

~~~java
ellipse(300, 200, 50, 80);
~~~

x = left/right
y = up/down
width = horizontal size
height = vertical size

**fill(r, g, b)**

Sets the fill colour for shapes.

~~~java
fill(255, 0, 0); // red
fill(0, 255, 150); // teal
~~~

**random(max)**
Generates a random number.

~~~java
fill(random(255), random(255), random(255))
~~~

**text(message, x, y)**

Draws text on screen.

~~~java
text("Hello!", 50, 100);
~~~

message = what to write
x, y = position

To set text size:

~~~java
textSize(32);
~~~
